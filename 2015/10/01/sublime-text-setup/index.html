Sublime Text 3的安装与简单配置（插件）

前言：由于公司里装的是Sublime Text 3（后文简称ST3），有强迫症的我，只好把家里升级为ST3了，也顺便将安装过程纳入Markdown之旅的第一篇博文吧。

如果不行上述方法安装失败，还可以下载Package control 离线包进行安装。在Manual下找到下面一行字，点击Package Control.sublime-package进行下载。

3.Download Package Control.sublime-package and copy it into the Installed Packages/ directory

下载完后如果不知道Installed Packages/在哪，可以按以下方式进入，Preferences > Browser Packages...，在打开的文件夹中，返回上一层Sublime Text 3，即可看到Installed Packages，进入后将下载的Package Control.sublime-package文件粘贴进去，重启ST3就会自动安装Package control。

有了Package Control就可以开始下载自己想要的插件了，网上关于ST系列的插件推荐有很多，我这里只记录我安装的，由此ST3只作为Markdown写作的编辑器，因此重装后我只安装了部分插件，下图为我安装的插件列表。

  1. Alignment 等号对齐，默认快捷键为ctrl+alt+a
    有强迫症的我，怎么能不用这种等号对齐的神器呢！
  2. Bracket Highlighter 高亮显示匹配的括号、引号和标签。
    在web开发的过程中，多层标签嵌套时有没有傻傻分不清哪个是哪个，这个插件可以让你快速理清各标签的嵌套关系。
  3. Git 你懂的，这个不说了。
  4. JsFormat 顾名思义，js的代码格式化插件，默认快捷键为ctrl+alt+f。
  5. Markdown Editing 主要用来Markdown语法高亮和替换主题。
  6. Pretty JSON json字符串的格式化，可以将压缩的json字符串格式化展开（快捷键ctrl+alt+j），也可以将展开的json字符串压缩（快捷键ctrl+alt+m）。

  7. Side Bar 侧边栏增强工具，ST原有的侧边栏功能太少了，这个可以扩展功能。
    这个其实是SideBarEnhancements，不知道为什么安装完名字变成这个了。
    安装完后可以添加快捷键，以便于快速将当前的文件用浏览器打开。
    进入Preferences > Packages Settings > Side Bar > Key Bindings - User，将下面代码粘贴进去。由于我主要使用chrome和firefox进行开发，因此此处只有2个配置，请自行修改浏览器的安装路径。

  8. Table Editor 据说Markdown的表格书写体验不好，有人专门开发了这个插件，还没体验过。

  9. Trailing Spaces 显示行末尾空格，适合强迫症删掉行尾空格。
    当然也可以在配置中将trim_automatic_white_space设置为true让ST3自动删除行尾空格。
  10. CSScomb css语句顺序调整，也可以自定义排序规则。
    对新手学习css有巨大帮助，可以让你知道规范的css语句顺序是怎样的，有助于思考如何布局一个页面
  11. ColorPicker 调色盘，谁用谁知道。
  12. OmniMarkupPreviewer 在实时浏览器中预览，谁用谁知道。
    妈妈再也不用担心写Markdown时代码预览的问题了。

ST3还有许多有用的插件，大家可自行下载使用，只是要注意插件之间的快捷键冲突问题。

遇到快捷键冲突记得在Key Bindings - User中修改。
